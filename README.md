# HKUST Navigation Drone

A self-learning project to build a autonomous navigation drone for HKUST campus both inside and outside.

**Started**: Feb 12, 2026

**Current state**: Phase 1

📁 [General Plan](docs/PLAN_4Phases.md)

📈 [Weekly Progress Log](docs/PROGRESS.md)

**Project description:**

As HKUST has complicated structure both vertically and horizontally, it is always
inconvenient to find the way of one location to another. Though HKUST Path Advisor is
commonly used, it can only provide verbal descriptions of the path and untransparent
visualized routes for users, which often lacks the intuitive, spatial context needed, leading
to confusion.

This project proposes an autonomous aerial guide – a drone capable of visually localizing
itself and physically leading users to their destinations.
To manage the project’s multidisciplinary complexity, I divided this project into four
phases: The first phase is to learn basic control and visual techniques and build up drone.
The second phase is to train drone to locate itself on campus with combined information
provided by visual module and supplemented sensors. The third phase is to train the drone
to move from one node to another and learn to avoid obstacles and go through different
kinds of entrances. The fourth phase is to run whole route and build up interaction system
with users.
